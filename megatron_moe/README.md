本项目为INF内部基于开源Megatron-LM框架进一步独立研发的大模型训练框架，持续跟踪社区（不限于Nvidia）的最新进展，同时基于自己的业务类型和集群特点持续补齐框架功能和优化训练性能。当前自研框架性能和功能满足度均优于开源Megatron-LM框架，支持训练、SFT、评估、Profiling等场景应用。
